## 📌 Overview
This project presents a research study on **sentence relationship understanding using BERT (Bidirectional Encoder Representations from Transformers)**.

Unlike traditional NLP models, BERT captures **bidirectional context**, allowing it to understand deep semantic relationships between sentences.

💡 This enables better performance in:
- Question Answering  
- Natural Language Inference  
- Sentence Similarity  
- Text Classification  

---

## 🧠 What Makes This Project Special?
✨ Focus on **Next Sentence Prediction (NSP)**  
✨ Implementation of **Masked Language Modeling (MLM)**  
✨ Hands-on training using **PyTorch + Transformers**  
✨ Covers both **theory + practical implementation**  

---

## 🏗️ Architecture Overview

Input Text → Tokenization → Embeddings → Transformer Encoder Layers → Output → Self-Attention Mechanism


✔ Uses **Bidirectional Transformer Encoder**  
✔ Captures **context from both left & right**  
✔ Generates **contextual embeddings**

## 🧪 Experiments

### 🔬 MLM Training
- Random tokens masked  
- Model predicts missing words  
- Learns deep contextual meaning  

### 🔬 NSP Training
- Sentence pair classification  
- Predicts whether sentence B follows sentence A  

### 🔬 SWAG Fine-tuning
- Commonsense reasoning task  
- Selects most logical sentence continuation
